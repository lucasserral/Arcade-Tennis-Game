Tennis Arcade Game

This is a basic arcade tennis game.

Wins the one that scores 5 times first.
When this happens, the game will be paused. You can begin a new game by clicking anywhere.

Future features:
  • The player will be able to change the speed of the ball.
  • The player will be able to choose the score to reach in order to win.
  • The game will be also be playable in mobile devices.
